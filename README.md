# Drogaria Sampaio’s – MySQL Database

A complete relational database for pharmacy management (clients, suppliers, products, invoices, payments).

**Highlights**
- 3NF modeling, foreign keys, checks, enums
- Sample data
- Views for reporting (`v_listagem_facturas`, `v_saldo_clientes`)

**How to use**
- Run `sampaio_database.sql` in MySQL (Workbench or CLI)

**Tech**: MySQL (DDL, DML, constraints, views)
